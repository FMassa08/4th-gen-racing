# 4th Gen Racing Standalone Website

Standalone Next.js project for the 4th Gen Racing website.

This project is intentionally separate from FilippoMassaRacing.com.

## Local Preview

```bat
cd /d "C:\Users\lexmi\Documents\4th-gen-racing"
npm run dev -- -p 3001
```

Open:

```text
http://127.0.0.1:3001
```

Or double-click:

```text
start-preview.cmd
```

## Build

```bat
cd /d "C:\Users\lexmi\Documents\4th-gen-racing"
npm run build
```
