# Filippo Massa Image Assets

Place approved Filippo Massa photography in this folder using these filenames:

- `hero-main.jpg`
- `portrait.jpg`
- `open-wheel-action.jpg`
- `closed-wheel-action.jpg`
- `helmet.jpg`
- `paddock.jpg`
- `car-detail.jpg`

Recommended source files: high-resolution JPG images, ideally at least 2400px wide for hero and action images. Keep the subject clear with enough dark or neutral space for text overlays.
